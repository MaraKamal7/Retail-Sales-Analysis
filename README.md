Retail Sales Analytics Project

Overview

This project was developed as an end-to-end Retail Sales Analytics solution using MySQL and Power BI.

The objective was to transform raw retail data into a structured analytical model that enables management to monitor business performance, identify growth opportunities, and make data-driven decisions.

The project covers the complete analytics lifecycle including data modeling, SQL analysis, KPI development, dashboard design, and business insights generation.

---

Business Problem

The retail company operates across multiple regions and collects sales data from different systems and years. Management lacked a unified analytical view to monitor performance and support decision-making.

The project aims to:

- Build a trusted data model
- Track key business KPIs
- Analyze sales and returns performance
- Identify underperforming products and regions
- Generate actionable business recommendations

Data Modeling

A dimensional model was designed using:

Fact Tables

- Fact_Sales
- Fact_Returns

Dimension Tables

- Dim_Customers
- Dim_Products
- Dim_Calendar
- Dim_Region

The final model follows a Galaxy Schema (Fact Constellation) structure with shared dimensions connected to multiple fact tables.

2. Which products or regions underperform?
3. What is the impact of returns?
4. Where should management focus next quarter?

---

Key Insights

- Sales growth is primarily driven by a limited number of high-performing products.
- Several products generate significant revenue but also experience high return rates.
- Certain regions consistently outperform others in both sales and profitability.
- Underperforming regions require targeted sales and marketing initiatives.
- Reducing return rates can significantly improve profitability.

---

Recommendations

- Increase focus on high-margin products.
- Reduce product return rates through quality improvements.
- Invest in customer retention programs.
- Expand successful regional strategies.
- Improve performance in underperforming regions.
